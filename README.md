ts-express
====================

This is a TypeScript based express starter

## usage ##

**Clone the repo**

```bash
git clone https://github.com/leftstick/ts-express.git
```

**Install dependencies**

```bash
cd ts-express
npm install
```

**Launch express server**

```bash
npm start
```
